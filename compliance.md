Compliance Change Radar → Auto-ticket + Evidence Bundle (most “judge-friendly”)
What it does: Watches regulator/vendor-policy pages (often annoying to navigate), detects changes, diffs, summarizes impact, creates a Jira/Linear ticket, pings Slack.
Why it wins: immediate enterprise value + perfect for browser agents + easy wow demo.
Sponsor usage
Browser Use: navigate + capture text/screenshot behind weird flows
Convex: scheduled runs + store page snapshots + diff pipeline
Laminar: trace each run + show replay when something breaks
HUD: success metric (“detected change correctly”, “ticket created”)
Vercel: dashboard (“Watched pages”, “Diffs”, “Runs”)
Demo moment: “Page changed → within 20 seconds we have diff + memo + ticket + evidence screenshots + hash.”
